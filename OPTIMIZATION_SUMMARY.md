## GitHub Actions 兼容性

增量更新机制完全兼容 GitHub Actions 自动部署：

1. **首次部署**: 自动获取完整历史数据
2. **每日更新**: 自动增量更新（工作日19:00）
3. **数据持久化**: 历史数据文件会被提交到仓库
4. **性能提升**: 每日更新时间从10分钟降至1分钟以内

## 注意事项

1. **数据目录**: `data/` 目录会自动创建，无需手动创建
2. **异常处理**: 如果历史数据文件损坏，会自动回退到完整获取模式
3. **数据保留**: 只保留最近365天的数据，自动清理旧数据
4. **日期边界**: 增量更新从上次更新前一天开始，确保数据连续性
5. **去重逻辑**: 同一日期的数据会自动去重，保留最新的数据
